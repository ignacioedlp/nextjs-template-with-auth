# Template para Full-Stack NextJS

### Librarias

- NextJs 14
- Next-Auth
- Prisma ORM
- PostgreSQL
- Eslint
- Prettier
- Bcrypt
- NextUI
- Zod

### Paginas

- Auth
    - SignIn
    - SignUp
- Dashboard (Necesitas estar autenticado)
- Users (Necesitas estar autenticado y tener el Rol de Admin)

### API

- SignUp
- SignIn → With next-auth […nextauth]
- Users → (Necesitas estar authenticado)
